当前目录为JQuery UI的根目录，想使用JQuery UI可以有以下两种方式：

【方式1】：
在你的页面中引入jquery-1.5.1.min.js（UI依赖库，两种方式都必须引入）、jquery-ui-1.8.12.custom.min.js和jquery-ui-1.8.12.custom.css

优点：
引入文件少，看起来清晰，无需关注细节

缺点：
此种方式是全引入，包含了JQuery UI的全部脚本库和样式表，看文件的尺寸就知道比较重量级，如果你只用到Tab或者其他的库最好单独引入库文件和样式表



【方式2】：
只引入自己需要的库文件和样式表，以Tab举例

引入JS文件：
jquery-1.5.1.min.js（UI依赖库，两种方式都必须引入）
jquery.ui.core.min.js（核心库，必须）
jquery.ui.widget.min.js（界面库，必须）
jquery.ui.tabs.min.js（Tab库）
前两个JS文件为必须引入的依赖库，无论使用Tab还是Dialog或者其他都要引入

引入CSS文件：
jquery.ui.core.css（核心样式，必须）
jquery.ui.theme.css（主题样式，必须）
query.ui.tabs.css（Tab样式）
前两个CSS文件为必须引入的依赖样式表，无论使用Tab还是Dialog或者其他都要引入

优点：
资源文件尺寸最小化

缺点：
引入的文件过多，不好维护，复杂度高
